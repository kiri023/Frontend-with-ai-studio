# 📋 AI브로(AI-Bro) 프로젝트 개발 To-Do 리스트

소상공인 정책자금 맞춤 추천 서비스 'AI브로'의 고도화를 위한 단계별 작업 목록입니다.

---

## 🚀 1단계: 기능 안정화 및 UX 고도화 (Short-term)
- [ ] **에러 핸들링 강화**: Gemini API 호출 실패 시 재시도 로직(Exponential Backoff) 추가
- [ ] **채팅 입력창 개선**: 모바일 환경에서 가상 키보드가 올라올 때 입력창이 가려지지 않도록 뷰포트 최적화
- [ ] **스켈레톤 UI 적용**: 데이터 로딩 및 AI 분석 중 대시보드와 채팅창에 스켈레톤 화면 노출
- [ ] **공고 공유 기능 구현**: Web Share API를 이용해 카카오톡이나 문자로 공고 링크 공유
- [ ] **데이터 양 확충**: 현재 5개 수준인 Mock 데이터를 50개 이상의 실제 최신 공고로 업데이트 (`constants.tsx`)

## 🔍 2단계: AI 엔진 및 검색 강화 (Mid-term)
- [ ] **Google Search Grounding 적용**: 최신 뉴스 및 정책 변화를 실시간으로 반영하기 위해 Gemini 3 Pro 모델과 검색 도구 연결
- [ ] **Google Maps Grounding 도입**: 사장님의 위치 주변 소상공인 지원 센터나 은행 지점을 찾아주는 기능 추가
- [ ] **다국어 지원**: 외국인 소상공인을 위한 다국어(영어, 중국어, 베트남어 등) 추천 인터페이스
- [ ] **이미지 분석 기능**: 종이 공고문이나 사업자 등록증 사진을 찍어 올리면 텍스트를 추출하여 프로필에 자동 반영

## 📊 3단계: 관리 및 데이터 분석 (Long-term)
- [ ] **리포트 내보내기**: AI가 추천한 자금 내역과 체크리스트를 PDF 파일로 다운로드하는 기능
- [ ] **알림 서비스**: 저장한 공고의 마감 3일 전 브라우저 푸시 알림 발송
- [ ] **통계 대시보드**: 내 업종의 평균 대출 금리나 인기 있는 정책자금 트렌드 시각화
- [ ] **백엔드 연동**: LocalStorage의 데이터를 서버 데이터베이스(PostgreSQL 등)와 동기화하여 기기 변경 시에도 데이터 유지

## 🎨 4단계: 디자인 및 퍼포먼스 (Optimization)
- [ ] **다크 모드(Dark Mode) 지원**: 야간에 업무를 보는 사장님들을 위한 시력 보호 테마 추가
- [ ] **접근성(A11y) 강화**: 시각 장애인을 위한 스크린 리더 최적화 및 고대비 테마 검토
- [ ] **성능 최적화**: 대량의 공고 리스트 렌더링 시 가상 스크롤(Virtual Scroll) 적용
- [ ] **PWA(Progressive Web App) 전환**: 앱스토어 설치 없이 홈 화면에 아이콘을 추가하여 앱처럼 사용 가능하게 구현
